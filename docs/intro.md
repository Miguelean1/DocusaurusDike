# DIKË — Documentación

**DIKË** es una plataforma comunitaria para compartir, prestar y donar objetos. Permite a los usuarios publicar artículos que desean prestar, donar o intercambiar, facilitando la comunicación entre quienes buscan y quienes ofrecen.

La aplicación incluye funcionalidades como:
- Publicación y gestión de artículos
- Sistema de solicitudes y reservas
- Mensajería directa entre usuarios
- Sistema de valoraciones y reputación
- Panel de administración

## Tecnologías principales

| Capa           | Tecnología                        |
|----------------|-----------------------------------|
| Frontend       | React 19.2.0 + Vite + Tailwind CSS |
| Backend        | Express.js 5.2.1 + Node.js        |
| Base de datos  | MySQL 8+                          |
| Almacenamiento | Cloudinary (imágenes)             |
| ORM            | Sequelize 6.37.8                  |
